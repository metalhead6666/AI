The package contains some examples for the use of the JCLAL Framework.

In "examples" folder there are many configurations files (*.cfg) that represents different experiments. To run any configuration file just type java -jar JCLAR.jar -cfg="path-to-cfg". If you want to run all the configuration files that there are into a folder just run java -jar JCLAL.jar -d="path-to-folder".

To see all the arguments supported by JCLAL.jar type java -jar JCLAL.jar.
